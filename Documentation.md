# 3D Portfolio — Documentation

## Overview

This project is a 3D-themed portfolio UI that highlights work, skills, and contact information with an interactive feel.

## Features

- Interactive/3D UI sections
- Content sections (Projects, Skills, Contact)
- Responsive layout

## Getting Started

### Install

```bash
npm install
```

### Run

```bash
npm start
```

or

```bash
npm run dev
```

## How to Use

1. Start the dev server.
2. Open the local URL in your browser.
3. Navigate through the sections to view portfolio content.
